自分のために、誰かのために。
コーヒーを淹れるって、しあわせ。

たっぷりとお湯を沸かし
豆が膨らむのを確かめながら、
細く、ゆっくりとコーヒーを落としていく。
ちょっと手間はかかるけれど、
こんなにも楽しめてしまうのは、
きっと、飲む人のことを想っているから。
おいしいコーヒーで、のんびりしたいな、
とか、
あの人には、心からくつろいでほしいな、
とか。
今日も、想いを巡らせながら。
「コーヒー、淹れたよ。」
毎日に、スペシャルなひととき。

田丸麻紀の
コーヒー、淹れたよ。とは？

すてきなコーヒーライフを楽しむ田丸麻紀さんとご近所の馬場園さんがおうちにさまざまな「おけいこ」の先生を招いて、コーヒーとともにおうちでおけいこを楽しむ番組です。

『おけいこ』の他にも、世界のコーヒーの楽しみ方を紹介するコーナー『世界のコーヒーブレイク』や、有名シェフがコーヒーに合う料理を紹介するコーナー『シェフズ・リコメンド』、コーヒーにまつわる心温まるお話を紹介するコーナー『コーヒーのある風景』などコーヒーがもっと楽しくなるコンテンツが目白押しです。

10月よりCSファミリー劇場で放送！
